## ⚡ The Mission

> Most ML engineers stop at the model.  
> I start *after* the model.

I build the systems that take a trained model and make it **do something real** — fast inference, resilient pipelines, agents that act, APIs that scale.

My domain is the **gap between research and production** — where beautiful math meets brutal reality.

If it doesn't ship, it doesn't count.
